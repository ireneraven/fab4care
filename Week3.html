    EDITING THE BODIES OF THE 3D SCANNING

    MESHMIXER

    Meshmixer is used, among other things, to clean up a 3D scan.

    The last week, we scanned our own body, scans which would be clean up by this software this week.

    Firstly, it is neccesary to import the scan you want to modify. In my case, I choosed the arm. Always, it is very useful for delate all the details you don't want in your scan, to press "select", select everything you would like to clean up, and press "edit". For ending, the option "Discard" should be pressed. (It is also possible just to select what you want to delate, and press the letter "X" on the keyboard).

    -->

    If it is desired to change the position of the object, the botton edit should be pressed, follow by the botton "transform".

    Another useful tool are the brushes which can be found pressing "sculpt". They are used to modify the 3D scan, improving the figure the way you want.

    As it can be apreciated, a wide variety of tools can be used. To finish, just mention that to close the figure completely, the option of "make solid" within "Edit" exists.

    Once all these steps are completed, the object has too be exported since now we are going to use the software Autodesk Fusion 360. This software was explain in the Week1.

    AUTODESK

    To start with, it is necessary to positionate in a properly plane the object, to make easier all the following process. The goal is to make a mold of our scan.

    Since the object is uploaded up to Autodesk, the sculpt mode will be the one that is worked on.

    Firstly, a cylinder must be design around the object. For that, it is necessary to prss "Create""-->"Cylinder".

    After this, the cylinder should be modify getting closer to the original object, making the desired mold. This can be done pressing "Edit form" withing "Modify".

    Once we have the mold, we hide the arm, and we just work with the mold. To continue, the mode has to be changed to "Model". Subsequently, within "Sketch"-->"Spline"-->"Fit point spline" creating we create a separatory line in the midle of our mold and pressing "modify" --> "Split body" we devide our figure in two different bodies.

    The last step in this program is to export the two bodies to Exact Flat. For that. firstly it's necessary to create the pieces as shown in the picture below. After this, we just have to upload them to the new program.

    EXACT FLAT

    Exact flat is the leader program in 3D to 2D digital pattering.

    Fristly, in "Prepare", we have to press in "Adaptive Remesher" of "Remesher".This should be done with the standard mode.

    Later, we shoul go to "Flatten", and press "Pre-Flatten" --> "Fracture".

    Once we have done these steps, it is necessary to choose the lines we consider appropriate to improve the 2D figure as much as possible. The goal is to eliminate any spikes that appear. When the image is clean, the option "Finish" should be pressed. Below is a picture of the final result.
